# Pipebot

A Discord bot for supporting CGI film pipeline workflows.

## Getting Started

- Install [uv](https://docs.astral.sh/uv/) and ensure it is on your `PATH`.
- Run `uv sync` to create the local environment and install dependencies.
- Execute the bot with `uv run python pipebot.py`.

## Development

- Run type checks with `uv run mypy`.
- Lint and format with `uv run ruff check .` and `uv run ty check`.
